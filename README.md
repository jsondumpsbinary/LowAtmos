# 🎵 LowAtmos

**Music shaped by mood and atmosphere.**
LowAtmos is a modern web application that delivers personalized music recommendations by combining real-time weather conditions with user emotions.

---

## 📖 About the Project

LowAtmos explores the connection between **environmental conditions** and **human emotions** to create a contextual music experience.
By analyzing weather data alongside mood input, the application dynamically curates songs that align with both internal and external states.

---

## ✨ Key Features

* 🌍 **Real-Time Weather Integration**
  Fetches live weather data based on user-selected location

* 😊 **Mood-Based Personalization**
  Allows users to select their emotional state

* 🎵 **Smart Music Recommendations**
  Suggests songs based on combined mood + weather logic

* 🔍 **Search Functionality**
  Enables users to search songs dynamically

* 🎯 **Filtering Options**
  Filter songs based on category or mood

* 🔽 **Sorting Capabilities**
  Sort songs alphabetically or by popularity

* ❤️ **Favorites System**
  Save and manage favorite songs using localStorage

* 🌙 **Dark Mode Support**
  Enhanced user experience with theme switching

* 📱 **Responsive Design**
  Optimized for mobile, tablet, and desktop devices

---

## 🛠️ Tech Stack

### Frontend

* React.js (Vite)
* JavaScript (ES6+)
* CSS / Tailwind CSS (optional)

### APIs

* OpenWeather API — Weather data
* Deezer API — Music data

---

## ⚙️ How It Works

1. The user inputs a city and selects their current mood
2. The application fetches real-time weather data
3. A mapping system determines the appropriate music category
4. Songs are retrieved from the music API
5. Results are rendered dynamically in the UI

---

## 🧠 Concepts & Implementation

* React Hooks (`useState`, `useEffect`)
* API integration using `fetch()`
* Array Higher-Order Functions:

  * `map()` for rendering lists
  * `filter()` for searching and filtering
  * `sort()` for ordering data
* Component-based architecture
* Event-driven programming
* Local storage for persistence
